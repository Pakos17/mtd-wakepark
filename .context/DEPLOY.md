- **Hébergement** : Vercel (site statique)
- **Repository** : GitHub
- **Domaine** : www.mtdwakepark.com (à configurer)

## Workflow
1. Modifier le code en local
2. Tester en local (ouvrir index.html dans le navigateur)
3. Commit + push sur GitHub
4. Vercel déploie automatiquement depuis la branche `main`
5. Vérifier le déploiement sur l'URL Vercel
