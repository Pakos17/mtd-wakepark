# MTD Wake Park — Site Web

## Déploiement
1. Cloner ce repo
2. Remplacer les images placeholder dans `/assets/images/`
3. Configurer Formspree : remplacer `VOTRE_ID` dans le formulaire
4. Configurer l'iframe Google Maps avec les bonnes coordonnées
5. Déployer sur Netlify/Vercel (drag & drop du dossier)

## Structure
- `index.html` — Page unique (one-page)
- `css/styles.css` — Tous les styles
- `js/main.js` — Toutes les interactions
- `assets/` — Logo, images, icônes SVG
